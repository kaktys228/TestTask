# Инструкция по настройке и запуску приложения "Служба доставки"

## Установка

1. Склонируйте репозиторий на локальный компьютер.
2. Убедитесь, что у вас установлен .NET SDK (версии 5.0 и выше).
3. Создайте необходимые папки для хранения логов и результатов (например, на рабочем столе):
   - `C:\Users\1\Desktop\`
   - В этих папках будут созданы файлы логов и результатов фильрации.
## Настройка
1. Создайте файл `Orders.txt` в указанной вами директории (например, `C:\Users\1\Desktop\Orders.txt`) и заполните его данными в формате:
OrderId; Weight; District; DeliveryTime
Пример:
1; 2,5; Восточный округ; 2024-10-22 10:00:00 
2; 1,2; Западный округ; 2024-10-22 10:30:00
## Запуск

Для запуска приложения используйте следующую команду в командной строке:

dotnet run -- "<Название района>" "<Дата>" "<Путь к файлу с логами>" "<Путь к файлу хранения фильтрации>" "<Путь к файлу с данными о доставки>"
Пример:
dotnet run -- "Восточный округ" "2024-10-22 09:40:00" "C:\Users\1\Desktop\DeliveryLog.log" "C:\Users\1\Desktop\FilterOrder.txt" "C:\Users\1\Desktop\Orders.txt"
